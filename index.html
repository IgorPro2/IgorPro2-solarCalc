Help
  1. Input your: Latitude/Longitude and TimeZone difference from UTC.
  2. Input your: Date/Time and Temperature/Pressure (for correction SunHeight on vertical refraction). BUTTONS ARE:
  3. "CALCULATE"- Calculate apparent sun's position at given Place Date and Time.
  4. "CURRENT" - Calculate SunPosition each second at system clock Time.
  5. "GRAPH" - Draw and animate SunPathes: Year(analemma) and Day moves.
  6. "GivenHtAz"- Calculate local Time moment when Sun's Center is at your specified Height or Azimuth. The Altitude/Azimuth values you need, are set in the line below the BUTTONS. To calculate the Time when the Sun is at a given height in the afternoon, check the box in this line. If Sun's Height or Azimuth impossible at the specified Place/Date, their values are given in red.
